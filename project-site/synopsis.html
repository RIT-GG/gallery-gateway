Project Synopsis

Gallery Gateway is an application that allows for students to submit their portfolios to be judged in scholarship opportunities and honor shows hosted by the School of Photographic Arts and Sciences. The project is a continuation of a senior project that has been developed by three previous senior project teams.

This year, we will be working to finish development of unfinished features the previous teams were unable to complete and adding new features to the projectâ€™s backlog. The current version of the application supplied to the sponsor does not meet the requirements for a minimum viable product, and it will be our focus to add the functionality needed to meet that goal. Our first main priority will be to complete functionality for scholarship applications and portfolio judging, then move to features regarding honor shows.
